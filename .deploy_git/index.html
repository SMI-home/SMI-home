更多想象空间的发展最早可追溯到2015年龙马创客部落，近年来我社在北京邮电大学校内展开了工程科幻创客马拉松，概念探索等大量活动。社团的宗旨是播种工程想象力，为全人类发展提供持续的前瞻概念探索！社团的使命是让每个人都能拥有一次属于自己的工程科幻畅想和创作机会，看到自己的思维。让工程想象力点燃内心的火焰，照亮人生，照亮人类社会与整个世界持续发展的道路。社团的愿景是发展面向未来的工程思维，使社团成为工程师创新灵感的采集地和激发地，成为工程科幻者的交流、沟通、思维碰撞的思想乌托邦。更多想象空间欢迎您的加入！
